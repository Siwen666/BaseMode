# BaseMvp
Android构建MVP（Model View Presenter）模型项目架构，资料太多就不献丑了。简单看下我的分包结构好了。

- View 对应于Activity，负责View的绘制以及与用户交互
- Model 依然是业务逻辑和实体模型
- Presenter 负责完成View于Model间的交互
